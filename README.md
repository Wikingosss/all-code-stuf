# all-code-stuf
> Short one-liner: what this repo does.

## 📦 Stack
- Language(s): Python 3.11
- Framework: FastAPI
- DB: SQLite (dev) / Postgres (prod)

## 🚀 Quick start
```bash
git clone https://github.com/wikingosss/all-code-stuf.git
cd all-code-stuf
python -m venv .venv
source .venv/bin/activate  # or .venv\Scripts\activate on Windows
pip install -r requirements.txt
uvicorn main:app --reload
